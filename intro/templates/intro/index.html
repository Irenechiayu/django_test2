Intro to Django Examples

We have three examples (four if you count this index.html page):
  1. Hello World With No Template -- This is a most basic hello world example, that simply sends back an HttpResponse object containing the HTML we want to provide in the response.
  2. Hello World (with template) -- A very basic example. The action (in views.py) immediately renders a templated view with empty context. The view displays Hello World.
  3. hello.html -- This example checks for a name parameter in the GET request. If present, it puts the name into the context and renders a templated view which says hello to the name. It (then) prompts for a(nother) name.
